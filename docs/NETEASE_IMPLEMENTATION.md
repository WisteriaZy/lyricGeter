# 网易云音乐 API 实现总结

## 概述

成功实现网易云音乐歌词获取，支持混合格式（JSON 元数据 + 标准 LRC）和翻译。

### 实现文件

### 1. 加密模块
**`decryptor/eapi.py`**
- EAPI 加密：AES-ECB + MD5 签名
- 遵循网易云官方加密协议
- 支持任意参数字典加密

### 2. 解析器（3 个）
**`parser/netease_word.py`** ⭐ 新增
- 网易云专有逐字格式解析
- 正则匹配 `[ms,ms](ms,ms,0)字`
- 提取每个字的时间戳和持续时间

**`parser/yrc.py`**
- YRC 格式解析（旧版逐字格式）
- 纯正则匹配，无需解密
- 提取时间戳和逐字文本

**`parser/json_lyric.py`**
- 新版 JSON 格式解析
- 每行一个 JSON 对象：`{"t": 时间ms, "c": [{"tx": "文本"}]}`
- 格式检测和内容提取

### 3. API 客户端
**`fetcher/netease.py`**
- 搜索歌曲：`/eapi/cloudsearch/pc`
- 获取歌词：`/eapi/song/lyric/v1`（含 `yrc`, `lrc`, `tlyric` 字段）
- 翻译作为独立字段传递

### 4. 格式转换
**`converter.py`**
- 优先检测网易云逐字格式 → `_netease_word_to_spl()`
- 支持混合格式：优先提取标准 LRC 行，过滤 JSON 元数据
- YRC → SPL：逐字时间戳转换
- JSON → SPL：行级时间戳转换
- 翻译行合并

### 5. CLI 集成
**`main.py`**
- 网易云 API 优先于 syncedlyrics
- 新增 `--no-netease` 选项禁用
- 默认启用，作为主要歌词来源

## 🎯 网易云逐字歌词格式

### 原始格式（网易云专有）
```
[13860,2310](13860,260,0)ヤ(14120,180,0)リ(14300,150,0)タ...
```

**格式说明**：
- `[起始时间ms,持续时间ms]` - 行级时间戳
- `(时间ms,持续ms,0)字` - 每个字的时间戳
- 这是**网易云专有格式**，不是标准 LRC

### SPL 转换结果
```
[00:13.86]ヤ[00:14.12]リ[00:14.30]タ[00:14.45]カ[00:14.67]ネ[00:14.99]ー...
```
- 每个字都有独立的结束时间戳
- 符合 SPL 逐字规范
实际测试发现，网易云 API 返回**三种格式**：

**格式 1：逐字格式（最优）**
```
[13860,2310](13860,260,0)ヤ(14120,180,0)リ(14300,150,0)タ
```

**格式 2：混合格式（JSON + LRC）**

```
{"t":0,"c":[{"tx":"作词: "},{"tx":"ANU",...}]}   # JSON 元数据
{"t":982,"c":[{"tx":"作曲: "},{"tx":"ANU",...}]}
[00:03.93]音乐人：旦旦                            # 标准 LRC
[00:11.82]ཚིག ཨ་ནུ་རིང་ལུགས།
[00:12.59]དབྱངས ཨ་ནུ་རིང་ལུགས།
...
{"t":259705,"c":[{"tx":"推广：麻雀音乐人"}]}      # JSON 推广信息
```

**格式 3：纯 JSON 格式**
```json
{"t":21460,"c":[{"tx":"飞"}]}
{"t":22700,"c":[{"tx":"当你想展翅翱翔之时"}]}
```

**解决方案**：
- 优先检测逐字格式 → `parser/netease_word.py`
- 然后检测混合格式 → 提取 LRC 行，过滤 JSON
- 最后处理纯 JSON → `parser/json_lyric.py`

### 翻译处理
- `tlyric` 字段独立返回，格式为标准 LRC
- 在 `converter.py` 中按时间戳匹配主歌词和翻译
- 翻译行紧跟主歌词后，无时间戳（符合 SPL 规范）

## 性能

- 搜索 + 获取歌词：约 7 秒
- 单文件完整流程：约 8 秒
- 网络请求：2 次（搜索 + 歌词）

## 已知限制

1. **逐字覆盖率**：约 42%，其余为行级歌词
2. **混合格式处理**：需要额外逻辑过滤 JSON 元数据
3. **相似度评分**：暂时固定为 100，待实现搜索结果元数据提取后恢复
